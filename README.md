# CodeAlpha - AI Chatbot

## 📌 Project Overview

A simple Java chatbot developed using rule-based responses.

The chatbot interacts with users through the console and responds to common queries.

## 🚀 Features

- Interactive Chat
- Rule-Based Responses
- Java Console Application
- Beginner Friendly

## 🛠 Technologies

- Java
- Scanner
- String Handling

## ▶️ Run

Compile

```
javac AIChatbot.java
```

Run

```
java AIChatbot
```
